## ⚠️ 强约束（发布阻断）

1. ✅ **请求体必须由 `_build_payload()` 统一生成**
2. ❌ **禁止 direct payload override**
3. ✅ **同步模式下，提交后必须等待结果或明确超时**
4. ✅ **超时时间 600 秒，轮询间隔 20 秒**
5. ✅ **参考图数量限制**：MJ≤3 张，atmosphere≤1 张
6. ❌ **不再承诺"后台自动通知"**

## 🚀 快速使用

### 配置检查
```bash
cd /home/ec2-user/.openclaw/skills/airilab
python3 scripts/check_config.py
```

### MJ 创意渲染
```bash
python3 core/api.py --tool mj --prompt "现代建筑外观，玻璃幕墙"
```

### 创意放大
```bash
python3 core/api.py --tool upscale --base-image "https://example.com/image.jpg"
```

### 氛围转换
```bash
python3 core/api.py --tool atmosphere \
  --base-image "https://example.com/image.jpg" \
  --prompt "转换为夜晚氛围，暖色灯光"
```
